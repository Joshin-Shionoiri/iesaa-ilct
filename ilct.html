国際学習能力検定（ILCT）

学びの力を、世界基準で測る。
成績や偏差値だけでは見えない、
「考える力」「理解する力」「伝える力」を、ひとつの指標に。

検定概要

国際学習能力検定（ILCT）は、知識量だけでなく、
理解力・思考力・表現力・問題解決力といった「学びの土台」を総合的に評価する検定です。
年齢や学年に依存せず、学習者一人ひとりの「今の学びの力」を可視化することを目的としています。

グレード構成
G1〜G6の6段階

G1：入門 / G2：基礎 / G3：標準
G4：中級 / G5：上級 / G6：最上級

対象
小学生〜社会人まで

学年ではなく「学習力のレベル」で受験グレードを選択できます。
塾・学校・個人受験いずれにも対応。

受験料
誰もが受けやすい価格設定

G1：1,500円 / G2：1,800円 / G3：2,200円
G4：2,700円 / G5：3,300円 / G6：3,900円

受験の流れ

  • 申込
    特設サイトからオンラインで申込。団体申込にも対応。
  • 受験票発行
    メールまたは所属団体経由で受験票を配布。
  • 試験当日
    安全・衛生に配慮した会場で試験を実施。
  • 成績通知
    学習力プロファイルとして、強みと課題をフィードバック。
  • 次回受験・学習支援
    次に受けるべきグレードや学習の方向性を提案。

サンプル問題

ILCTでは、「正解を覚える力」ではなく、「考えるプロセス」を重視した問題を出題します。
以下は一例です。

例：G3レベル
文章を読んで、要点を整理する問題

ある文章を読み、「筆者が一番伝えたいこと」を一文でまとめる問題など。
読解力と要約力を同時に測ります。

例：G5レベル
図表を読み取り、自分の考えを述べる問題

グラフや表の情報を整理し、「そこから言えること」を論理的に説明する問題。
分析力・論理的思考力・表現力を総合的に評価します。
